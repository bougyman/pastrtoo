Welcome to Pastr

Pastr is an interactive help system bringing the web and Internet Relay Chat together.

Our goal is to supply a system where IRC users seeking assistance can request a link from the Pastr irc bot and be able to paste text to that link. The text can be syntax highlighted into any of our supported languages (115 to date) if you choose a language when pasting/editing. The IRC bot will relay the link for others to view/edit your post to the IRC channel (or nickname) who you are seeking assitance from. Each post is revision controlled, so edits do not overwrite, they make a new version.

Features

  1. Pastr bot on Freenode and Efnet, responds to .hitme in public or private to provide a link
  2. Pastr website, to edit and view pastes
  3. Syntax Highlighting for 115 different languages (and configuration files)
  4. Plain text, auto-filename view. If the post view is http://paste.linuxhelp.tv/100, then you may use a tool such as 
    wget http://paste.linuxhelp.tv/100/grub-install.txt
    to download the plain text contents of the paste as a file locally. The filename you choose is arbitrary, so long as it starts with a word (/^\w/)

Hints

  • xclip is a great helper for pasting text files. 
    xclip < /etc/fstab
    Would read the contents from /etc/fstab into your X paste buffer. Then you just middle click on the paste body to paste /etc/fstab to Pastr.
  • How do I just make a new paste on the web? You don't. This system is currently only available to users of Efnet or Freenode IRC via the Pastr nickname on them
